## Setup

After cloning run `git submodule update --init --recursive`

If running Linux make sure all these are installed `wayland-protocols`, `libgl1-mesa-dev`, `libwayland-dev`, `libxkbcommon-dev`, `libxrandr-dev`, `libxinerama-dev`, `libxcursor-dev`, and `libxi-dev`.
